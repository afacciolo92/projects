## Credit Default Compass

Predictive modelling of credit card defaults using the UCI Credit Card dataset. Includes EDA, feature preprocessing, and comparison of Logistic Regression and Random Forest classifiers with hyperparameter tuning.

**Tools:** Python, scikit-learn, pandas, NumPy, Matplotlib, Seaborn

[Report](credit-default-compass/Credit%20Default%20Compass.pdf) | [Code](credit-default-compass/Credit_Default_Code.py)
